## Requirements

- PHP 5.6 or higher
- cURL
- Twitter Developer Account
- Twitter App

## Get started

### Installation

I strongly advice using [Composer](https://getcomposer.org) to keep updates as smooth as possible.

```
$ composer require espresso-dev/twitter
```
